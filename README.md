# QR Code Attendance System

## 📌 Giới thiệu

Hệ thống điểm danh bằng QR Code được xây dựng bằng ASP.NET Core MVC.

## 🚀 Chức năng chính

* Đăng nhập người dùng
* Quản lý người dùng
* Tạo buổi điểm danh
* Sinh QR Code
* Check-in bằng QR
* Xem lịch sử điểm danh

## 🛠 Công nghệ sử dụng

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* QRCode (QRCoder)
